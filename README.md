# ManaKit

A GraphQL implementation of [Scryfall](http://scryfall.com/).

## Usage

ManaKit is now a GraphQL data graph. 


## Graphql Code Generation

Note that this recreates the Package.swift, and need to be reverted.

`./apollo-ios-cli generate --fetch-schema`

## Installation

ManaKit is available as a Swift Package Manager: https://github.com/vito-royeca/ManaKit.git
